# ETB — Expand the Box

A complete, self-paced digital adaptation of the Expand The Box training from Possibility Management. This repository holds the course versions side by side.

> **Honesty note:** this digital course is a derivative work created in the spirit of [World Copyleft](https://worldcopyleft.mystrikingly.com/); it is not an official Possibility Management offering. The lineage is Possibility Management (Clinton Callahan and the PM trainer network). See each version's `LICENSE.md`.

## Versions

| Folder | Version | Status |
|---|---|---|
| `Digital Expand The Box May 2026/` | v1 — the original 10-day digital course | **Frozen.** Reference only; never modified. |
| `Digital Expand The Box June 2026/` | v2 — the 2026 rebuild (12 modules, practice container, HTML shell) | **Active build.** See `_build/HANDOFF.md` for build state. |

**Learners:** open the June 2026 folder and start at `START HERE.md` (or `index.html` once the shell ships).

**Operators / contributors:** the June 2026 `README.md` is the orientation. The build plan is `_build/ARCHITECTURE.md` (locked), built from `_build/AUDIT.md` (an 11-dimension audit of v1). The Voice and Style Guide in `Facilitator Resources/` is binding for all contributions.

## Why two versions

v2 is the same course taught better: v1's teaching content, safety architecture, and voice carry forward largely intact, while v2 adds a real front door and HTML delivery shell, a daily practice spine wired into every module, a retrieval layer inside the no-quiz stance, canon and safety patches, and the operator/commerce/continuation layer — plus a sanctioned solo path. The full rationale lives in the June 2026 `_build/AUDIT.md`.
