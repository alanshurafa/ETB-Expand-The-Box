# Map 44 — Linear and Nonlinear

*Two kinds of possibility that are not two sizes of the same thing — linear extends the current context by ordinary steps; nonlinear becomes available only when the context itself changes.*

![Linear and Nonlinear — Possibility Management map](../Maps/M44.png)

**What it is.** Two kinds of possibility, and they are not two sizes of the same thing. Linear possibility extends what already exists: more clients, a better apartment, a clearer version of the conversation you already have. It is reachable by ordinary steps from the current context, and most planning, goal-setting, and self-improvement operates entirely inside it. Nonlinear possibility is not reachable by extension at all. It becomes available only when the context itself changes: a declaration, a dangerous question, a context shift that makes a different set of next actions visible. From inside the old context, nonlinear possibility looks like nothing, which is exactly why the Box never objects to it in advance. Linear gets you a better seat in the room you are in; nonlinear changes which room exists.

**At a glance.** Linear → extends the current context; reachable by ordinary steps; the domain of plans · Nonlinear → available only when the context changes; reached through declaration and dangerous questions, not extension · From inside the old context, nonlinear looks like nothing → which is why the Box never objects to it in advance · Linear gets a better seat; nonlinear changes which room exists · Neither is superior in every domain → renewing your passport is a linear job; do not declare at it · Every result you have already concluded is impossible lives, if anywhere, on the nonlinear side.

## How to explain it verbally

There are two kinds of possibility, and they are not big and small versions of one thing. Linear possibility extends what already exists: more clients, a better apartment, a cleaner version of the conversation you already keep having. You get there by ordinary steps from where you already stand, and almost all planning and goal-setting lives here. Nonlinear possibility is not reachable by extension at all. It only shows up when the context itself changes, through a declaration or a dangerous question. From inside the old context it looks like nothing, which is exactly why the Box never objects to it in advance. Linear gets you a better seat in the room you are in. Nonlinear changes which room exists. Neither is superior everywhere; renewing your passport is a linear job, do not declare at it. But anything you have already filed as impossible lives on the nonlinear side.

**If you only remember one thing:** linear gets you a better seat in the same room; nonlinear changes which room exists, and it only opens when the context shifts.
