## 4. Video Manifest

Modules are text-first and complete without video. Source videos are optional enrichment, registered (filename → status → where-to-get) in [`Facilitator Resources/Video Manifest.md`](Facilitator%20Resources/Video%20Manifest.md). No learner-facing file references a bare video filename.

## 5. Link conventions

1. Relative paths only; spaces percent-encoded (`%20`) in every link target.
2. Learner-facing files link only paths that ship in this folder. No `Possability Lab`/`SPARKS` source-library paths, no v1 paths, no bare `.mp4` filenames.
3. Markdown is the canonical source; the build pipeline rewrites `.md` links to `Course/*.html` for the shell.
4. The automated link checker (`_build/`) is a release gate: zero broken internal links.
